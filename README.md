## Pick the right tool
**Workspaces**: [yarn workspace](https://classic.yarnpkg.com/lang/en/docs/workspaces/)

**CSS Styling**: [https://tailwindcss.com/](https://tailwindcss.com/)
**UI Libraries**: [React](https://reactjs.org/) + [Storybook](https://storybook.js.org/)
